## Группы сигналов

Даже без полного pin assignment в даташите указаны основные группы интерфейсов, которые важно учитывать при трассировке и подборе обвязки:

- **Критичные к трассировке высокоскоростные группы (дифф‑пары)**:
  - HDMI 2.1 (до 8K/60)
  - eDP 1.4 (до 8K/30)
  - MIPI‑DSI / MIPI‑CSI (D‑PHY/C‑PHY — согласно параметрам в даташите)
  - PCIe Gen3/Gen2 (lane‑группы)
  - USB 3.x / Type‑C с DP Alt Mode (DP 1.4)
- **Питание и земля**:
  - основное питание модуля: 5 В (точные пины и распределение по разъёму — TODO)
- **Низкоскоростные/служебные группы**:
  - UART, I2C, GPIO, CAN
